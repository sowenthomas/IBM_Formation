1- Répondez aux questions suivantes :

		1. A quoi sert la déclaration DOCTYPE ? 
			La déclaration du DOCTYPE permet d'indiquer à quelle règle d'écriture obéit le code d'une page html.
			
		2. Comment on peut valider un document html ?
			On peut valider un document html avec w3c validator accessible via validator.w3.org .
			
		3. Quel emplacement est défini le jeu de caractères utilisé dans le document ?
			La balise meta avec l'attribut charset permet de définir le jeu de caratère utilisé dans le document.
			
		4. Quel est le rôle des balises sémantique?
			Le rôle des balises sémantique c'est structurer la page html afin de donner un sens aux informations qu'elles encadrent.
			Ces dernières permettent à améliorer le référencement sur les moteurs de recherche et de de permettent aussi aux personnes atteintes de déficience visuelle qui utilise un synthétiseur vocal, de lire les pages web.
